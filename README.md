A small Flask-based HR application to manage employee flags and departments. Employees have configurable departments, salary, bonus, and days off. The frontend displays all employees and allows setting department flags.

## Features

- View all employees with their active department, salary, bonus %, and days off.  
- Select an employee and set their department flag.  
- Departments are loaded from `departments.json`.  
- Simple, self-contained Flask app.

## Requirements

- Python 3.12+
- Flask
